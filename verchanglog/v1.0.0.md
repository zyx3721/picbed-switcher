## 📋 版本概述

v1.0.0 是 picbed-switcher 的首个正式版本，完成了图床转站助手的核心闭环：用户认证、图床配置管理、Markdown 图片地址分析、图片迁移上传、转换历史记录、前后端分离界面、Docker Compose 部署与 Swagger API 文档。

---

## ✨ 新增特性

- 新增基于 JWT 的用户注册、登录、个人资料查询能力。
- 新增当前用户密码与邮箱修改接口。
- 新增图床配置管理，支持新增、编辑、删除、默认配置设置与敏感配置加密存储。
- 新增 Markdown 文档图片地址分析、单文档转换与批量转换能力。
- 新增转换历史记录查询，便于追踪源图床、目标图床、状态和图片数量。
- 新增多图床上传适配，支持 GitHub、Gitee、腾讯云 COS、阿里云 OSS、七牛云、EasyImage 和兼容通用上传接口的图床服务。
- 新增 Vue 3 + TypeScript 前端工作台，包含登录注册、概览、图床配置、文档转换和历史记录页面。
- 新增 Swagger 接口文档，访问地址为 `/swagger/index.html`。
- 新增 Dockerfile、Docker Compose、Nginx 和 Supervisor 配置，支持一键容器化部署。
- 新增 CI 工作流和项目预览图片资源。
- 修复 Release Actions 中后端目录与附件路径配置，确保 v1.0.0 可正确构建二进制产物并发布 Docker 镜像。

## 🚀 性能优化

- 后端接口启用请求速率限制，降低高频请求对服务稳定性的影响。

## 🔄 API 变更

首个正式版本提供以下 API：

- `GET /health`：服务健康检查。
- `POST /api/auth/register`：用户注册。
- `POST /api/auth/login`：用户登录。
- `GET /api/auth/profile`：获取当前用户信息。
- `PUT /api/auth/password`：修改当前用户密码。
- `PUT /api/auth/email`：修改当前用户邮箱。
- `GET /api/picbed/types`：获取支持的图床类型与字段定义。
- `GET /api/picbed/configs`：获取图床配置列表。
- `POST /api/picbed/configs`：创建图床配置。
- `PUT /api/picbed/configs/:id`：更新图床配置。
- `DELETE /api/picbed/configs/:id`：删除图床配置。
- `PUT /api/picbed/configs/:id/default`：设置默认图床配置。
- `POST /api/convert/analyze`：分析 Markdown 文档图片地址。
- `POST /api/convert/process`：执行单个 Markdown 文档转换。
- `POST /api/convert/batch`：批量执行 Markdown 文档转换。
- `GET /api/convert/records`：获取转换历史记录。
- `GET /swagger/index.html`：Swagger API 文档页面。

## 📊 数据库变更

首个正式版本新增 PostgreSQL 初始化结构：

- `users`：用户账户表，保存用户名、密码哈希、邮箱与创建/更新时间。
- `picbed_configs`：图床配置表，按用户保存图床类型、配置名称、加密配置和默认配置状态。
- `conversion_records`：转换记录表，保存源文件、源图床、目标图床、转换状态、错误信息和图片数量。
- 新增用户、配置、转换记录相关索引和更新时间触发器。

## 🔄 升级说明

v1.0.0 为首个正式版本，无历史版本升级迁移要求。

全新部署时请执行 `backend/migrations/001_init_schema.sql` 或通过应用首次启动自动初始化数据库结构。生产环境请务必修改 `.env` 中的数据库连接、`JWT_SECRET` 与 `ENCRYPTION_KEY` 等敏感配置。
